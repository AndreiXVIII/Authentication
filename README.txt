index.php - Главная страница, на которой можно: авторизироваться, зарегистрироваться, 
	    перейти в админку, при условии того, что у пользователя есть на это соответствующие права, 
	    перейти в личный кабинет и завершить работу (сессию).

connect.php - Подключение к базе данным. Затем инклудится во все остальные файлы. Для избегания дублирования кода

login.php - Авторизация пользователя

register.php - Регистрация нового пользователя

logout.php - Пользователь перестает быть авторизированным, сброс активных сессий 

users.php - Показывает списком всех зарегистрированных пользователей

profile.php - Личный кабинет пользователя

personalArea.php - Редактирует личный данные пользователя

changePassword.php - Меняет пароль

deleteAccount - Удаляет аккаунт

admin.php - Админка. Удаления пользователя, присвоение статуса, бан.
